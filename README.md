# 作業二-21點連線版
##實作功能
* 可五位玩家連線
* 莊家發牌
* 玩家選擇是否加牌
* 分數結算

##程式說明與截圖
首先是一位玩家及一個莊家的模擬狀況(1 server-1 client)
client端需輸入兩個參數 -- host,port
